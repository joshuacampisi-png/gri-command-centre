# SEO Content Policy

**Effective:** 2026-03-18  
**Owner:** Pablo Escobot / Josh Campisi

---

## Rule: All SEO Content Must Be Reviewed by Educated SEO Agents

### What This Means

**Before any SEO content change goes live**, it must be:

1. ✅ **Generated by an SEO specialist agent** (not generic Claude)
2. ✅ **Validated against SEO best practices**
3. ✅ **Approved by Josh via Telegram**
4. ✅ **Logged with reasoning and target keywords**

### What Requires SEO Agent Review

- Meta descriptions
- Meta titles
- H1 headlines
- Product descriptions (when SEO-focused)
- Collection page copy
- Alt text for hero/featured images
- Schema markup content

### What Can Auto-Fix WITHOUT SEO Agent

- Alt text for generic product images (descriptive only)
- 404 redirects (technical, no content)
- Image optimization (file size, format)
- Broken link fixes

## Approval Flow (SEO Content)

```
1. Flywheel detects issue
   ↓
2. Consult appropriate SEO agent
   ↓
3. Agent generates fix + reasoning
   ↓
4. Validate response (length, keywords, quality)
   ↓
5. Send to Josh via Telegram for approval
   ↓
6. Josh replies /approve <taskId>
   ↓
7. Push to live theme
   ↓
8. Log to Notion with agent reasoning
```

## Quality Gates

### Meta Descriptions

- ✅ 150-160 characters
- ✅ Includes primary keyword naturally
- ✅ Location signal ("Australia" / "Australian")
- ✅ Action-oriented CTA
- ✅ Unique for each page
- ✅ SEO agent provides keyword reasoning

### Meta Titles

- ✅ 50-60 characters
- ✅ Brand name at end: "... | Gender Reveal Ideas"
- ✅ Primary keyword near start
- ✅ Compelling, click-worthy

### Alt Text

- ✅ Descriptive (accessibility-first)
- ✅ Under 125 characters
- ✅ Keywords natural, not stuffed
- ✅ Unique per image
